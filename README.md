# Health Monitor Dashboard

A lightweight web dashboard for real-time sensor monitoring.

## Overview
Displays live sensor data, device status, and recent activity in a simple interface.  
Designed for real-time visualization with minimal configuration.

## Setup
1. Clone the repository  
2. Create a local config file:
   ```bash
   cp config.example.js config.js
````

3. Add your connection details to `config.js`
4. Open `index.html` in a browser or serve locally

## Security

* `config.js` is not included in the repository
* Do not commit sensitive credentials
* Some system components are intentionally abstracted

## Status

Active development — documentation is limited by design.
